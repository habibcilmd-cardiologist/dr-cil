تصوير الأوعية القلبية هو تقييم أوعية القلب باستخدام التصوير بالأشعة السينية. إنها الطريقة القياسية الذهبية لتشخيص مرض الشريان التاجي. يتمتع أخصائي أمراض القلب التداخلية أ.م.د. حبيب جيل بخبرة واسعة في إجراءات تصوير الأوعية القلبية.

## ما هو تصوير الأوعية القلبية؟

تصوير الأوعية القلبية هو تصوير أوعية القلب باستخدام مادة التباين. يتم الكشف عن التضيق والانسداد والشذوذ في الشرايين التاجية.

**أغراض تصوير الأوعية القلبية:**

-   تشخيص مرض الشريان التاجي
-   تحديد درجة التضيق
-   تخطيط العلاج
-   قرار التدخل

## دواعي تصوير الأوعية القلبية

### الدواعي التشخيصية

-   الذبحة الصدرية المستقرة
-   متلازمة الشريان التاجي الحادة
-   اختبار جهد إيجابي
-   تقييم قصور القلب

### الدواعي التدخلية

-   تخطيط الدعامة
-   قبل جراحة المجازة
-   قبل جراحة الصمام
-   أمراض القلب الهيكلية

## التحضير قبل تصوير الأوعية القلبية

### تحضير المريض

-   صيام 6-8 ساعات
-   اختبارات وظائف الكلى
-   تعديل مميع الدم
-   الاستفسار عن تاريخ الحساسية

### موانع الاستعمال

-   فشل كلوي حاد
-   حساسية التباين
-   عدوى نشطة
-   ارتفاع ضغط الدم غير المنضبط

## كيف يتم إجراء تصوير الأوعية القلبية؟

### طرق الوصول

**النهج الفخذي:**

-   الوصول عبر الفخذ
-   الطريقة الكلاسيكية
-   يمكن استخدام قسطرة كبيرة

**النهج الكعبري:**

-   الوصول عبر المعصم
-   مضاعفات أقل
-   التعبئة المبكرة

### steps الإجراء

1. **التخدير الموضعي**: لموقع الوصول
2. **ثقب الشريان**: فخذي أو كعبري
3. **تقديم سلك التوجيه**: إلى الشريان الأبهري
4. **وضع القسطرة**: إلى فتحات الشريان التاجي
5. **حقن التباين**: للتصوير
6. **تسجيل الصور**: من زوايا مختلفة
7. **إزالة القسطرة**: نهاية الإجراء
8. **الإرقاء**: السيطرة على النزيف

### مدة الإجراء

المدة الإجمالية: 15-30 دقيقة

## نتائج تصوير الأوعية القلبية

### النتائج الطبيعية

-   شرايين تاجية ملساء
-   لا يوجد تضيق
-   تدفق طبيعي

### النتائج المرضية

-   تضيق الشريان التاجي
-   انسداد كامل
-   توسع الأوعية الدموية
-   شذوذ

### درجات التضيق

| الدرجة | معدل التضيق |
| ------ | ----------- |
| خفيفة  | <50% |
| متوسطة | 50-70% |
| شديدة  | 70-90% |
| حرجة   | >90% |
